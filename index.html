Article: The Memory Controller Wall — Intel FPGA vs Nvidia GPU

Engineers from Edgecotix developed an OpenCL-based memory bandwidth benchmark suite to analyze the efficiency of the memory controller on Intel FPGAs. The suite covers multiple memory access patterns, namely sequential access, 1D blocking, and 1.5D/2.5D overlapped blocking (N.5D refers to blocking N dimensions of an (N+1)D input and streaming the remaining one [3]). Moreover, it provides control over multiple compile-time and run-time parameters such as number of input/output buffers, kernel programming model (NDRange or Single Work-item), on-chip channels between read and write, vector size, block size, halo size, buffer padding, operating frequency, and interleaving.
